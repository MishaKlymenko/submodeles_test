# Echo Server

Простий TCP Echo сервер написаний на Node.js для тестування Git submodules.

## Опис

Цей проект містить базовий echo сервер, який приймає текстові повідомлення від клієнтів і відправляє їх назад з префіксом "ECHO:".

## Використання

### Запуск сервера

```bash
npm start
```

або

```bash
node server.js
```

Сервер буде запущено на `127.0.0.1:3000`.

### Запуск клієнта

```bash
npm run client
```

або

```bash
node client.js
```

Після підключення ви можете вводити повідомлення, які будуть надіслані на сервер і повернуті назад.

## Тестування з telnet

Ви також можете протестувати сервер за допомогою telnet:

```bash
telnet 127.0.0.1 3000
```
